# quantio-library

A warm, cozy library management system — like a real library.

## Design

Warm wood tones, parchment backgrounds, and Playfair Display serif headings evoke the feeling of a classic reading room. Dark walnut sidebar, amber/oak accents, sage greens, and crimson details give it an earthy, literary atmosphere.

Built with vanilla HTML/CSS/JS — no frameworks, no build tools. Open `index.html` in any browser.
